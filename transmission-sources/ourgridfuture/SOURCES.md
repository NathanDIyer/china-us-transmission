## Notes

- The "proposed unified set" in extracted summaries excludes rows with `Status` = `Complete` or `Terminated`.
- The `230kV+` summaries filter on `Maximum voltage (kV) >= 230`.
- RTO summary uses split membership from the `RTO intersected` field; miles can be counted in multiple rows when a segment intersects multiple RTOs.
- Reconciled voltage-class projection uses a strict no-double-count lower-bound formula:
  - `UnifiedLowerBound(class) = max(NERC(class), OGF(class), ISO(class))`.
- This `max()` merge is intentional:
  - `sum(NERC + OGF + ISO)` would double count overlapping projects already represented in multiple datasets.
  - `max()` gives a conservative lower-bound union by class that is reproducible from public files.
- ISO/RTO values in reconciliation are limited to class-explicit disclosures only (MISO 345/765, NYISO CHPE 320, PJM/SPP 765); low-confidence PDF text-mined candidates are not included in roll-up totals.
- Full row-level provenance for each reconciled class is in:
  - `extracted/reconciled_us_projected_miles_source_trace.csv`.
